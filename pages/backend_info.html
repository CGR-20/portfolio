Processor & Assembler Project

The microprocessor, built in C++, emulates the process of executing custom bytecode given to it via a bytecode file, which if written properly will execute a program.

The assembler, built in Python, translates custom assembly code into the bytecode that the processor can read and execute.

Both programs work together to allow users to write and execute programs in a low-level assembly language.

C++ Python Shell Git

Dynamic Bit Array Project

A program that implements a dynamic bit array data structure, allowing for efficient storage and manipulation of bits.

It supports operations such as setting, clearing, toggling, and querying individual bits, as well as resizing the array dynamically as needed.

This project demonstrates an understanding of low-level data manipulation and memory management in C++.
